# Shopping List App

A simple, clean shopping list web app built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies â€” just open `shopping-list.html` in your browser.

## Features

- Add items by typing and pressing **Enter** or clicking **Add**
- Check off items as you go
- Delete individual items
- Remove all checked items at once
- Item count and progress tracker
- Data persists in `localStorage` (survives page refresh)

## Usage

Just open `shopping-list.html` in any modern browser. No build step or server required.
